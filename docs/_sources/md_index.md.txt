# pyDXHR Documentation

Python-based tools for data exploration, extraction, and file-based modding of _Deus Ex: Human Revolution_. Primarily intended as a model exporter, until it grew into other things. 

This project is NOT affiliated with Eidos Interactive, Crystal Dynamics, the Embracer Group, or Square Enix.

- Usage instructions:
  - For the pre-compiled single-use binaries: [entrypoints](entrypoints.md)
  - For the Python package: [package](package.md)
  - For the command-line tools: [cli](cli.md)
- Build and development instructions: [development](development.md).
